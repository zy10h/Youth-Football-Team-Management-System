Youth Football Team Management System

1. Purpose
This application is designed to manage youth football teams, players, and coaches.

---

2. Features
- Player management (CRUD)
- Team management (CRUD)
- Coach management (CRUD)
- Automatic team assignment based on age
- Manual override for team assignment
- Jersey number validation (1–99, unique per team)
- Search and sorting
- Pagination
- Authentication (JWT)

3. Technologies
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
